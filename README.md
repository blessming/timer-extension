# crypto-timer-react
Chrome extension to get the cryptocurrencies prices.

## Installation
  1) Install the dependencies.
   ```
   yarn
   ```
  
  2) Build the react project.
  ```
  yarn build
  ```
  
  3) Chrome.
  
  4) Three dots on the top right corner.
  
  5) More tools.
  
  6) Extensions.
  
  7) Drop the "build" folder.
